# 简介

- 基于SpringBoot+Vue+ElementUI的个人健康平台

- 后端系统仅搭建至用户列表

- 文件目录结构

  ```
  ├─Helth-Platform
    ├─HealthPlatform # 后端工程代码
    ├─Vue-HealthPlatform # 前端工程代码
    ├─DataBase-SQL # 数据库的建表语句 请注意!!! 表名 user==easyuser
  ```
